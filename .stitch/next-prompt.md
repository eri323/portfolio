A striking, asymmetrical main landing page for a QA Analyst and Junior Developer's professional portfolio.

**DESIGN SYSTEM (REQUIRED):**
A restrained, cockpit-dense interface with confident asymmetric layouts and fluid spring-physics motion. The atmosphere is professional yet highly technical — like a modern IDE.
Colors: Canvas Dark (#09090B), Code Surface (#18181B), Terminal Silver (#F4F4F5), Muted Steel (#A1A1AA), Electric Cyan (#06B6D4).
Typography: Outfit for display/body, JetBrains Mono for technical details. NO INTER.
Anti-patterns: No pure black, no nested glow shadows, no centered heroes, no overlapping text on images.

**Page Structure:**
1. A top header with a clean logo ("JR {DEV} & [QA]") and simple navigation.
2. A left-aligned Hero section introducing the dual-role: "Quality Assurance + Full-stack Engineering", with a sophisticated code snippet graphic on the right half.
3. An asymmetrical grid displaying "Core Competencies" (e.g. Automated Testing, React Native, Manual QA).
4. A clean recent projects featured list using border-top dividers (avoid 3-column cards).
5. A footer with social links and contact CTA.
